How to use

Back to top
  1. Copy framework.js and framework.css into your project directory
  2. Add these lines to the head of your page. 
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/normalize/8.0.0/normalize.min.css">
<link rel="stylesheet" href="https://use.fontawesome.com/releases/v5.0.13/css/all.css" integrity="sha384-DNOHZ68U8hZfKXOrtjWvjxusGo9WQnrNx2sqG0tfsghAvtVlRW3tvkXWZh58N9jp" crossorigin="anonymous">
<link rel="stylesheet" href="framework.css">
  3. Add this line right before the end of thebody of your page. 
    <script src="framework.js"></script>

Usage: col-{breakpoint}-{size}
size can be anywhere from 1 to 12
breakpoint can be xs, sm, md, lg or xl

The grid can be divided into 12 equal columns. Applying col-xs-8 on a grid element extends it to the 3/4 of the width of the grid on xs sized screens (e.g mobile phones). Combining it with col-md-6 will make it so that it only covers half the grid width on md sized screens (e.g. tablets) but still covers 3/4 of the grid width on xs sized screens.

col-xs-8 and col-md-6 
<div class="row">
  <div class="col-xs-8 col-md-6 theme-aqua center">
    col-xs-8 and col-md-6
  </div>
</div>

Themes

Usage: Add theme-{color} class to the element.
Replace {color} with any preset color from the grid below.
ex. theme-navy sets an element's theme to navy blue.

Background

Usage: Add bg-{color} class to the element.
Replace {color} with any preset color from the grid below.
ex. bg-navy sets an element's background color to navy blue.

Text (with shadow)

Usage: Simply add the name of the color to the class of the element.
ex. blue changes an element's text color to blue.

Display properties

  • Block - d-block
  • Inline - d-inline
  • Inline Block - d-inline-block
  • Flex - d-flex
  • Hidden - hidden, d-hidden, or d-none

Float and clearfix

  • Left - float-left
  • Right - float-right
  • Clearfix - clearfix

Text alignment

  • Left - left
  • Right - right
  • Center - center

Flexbox properties

  • Justify content

    • Start - flex-start
    • End - flex-end
    • Center - flex-center
    • Space around - flex-space-around
    • Space between - flex-space-between

  • Flex direction

    • Row - flex-direction-row
    • Column - flex-direction-column

  • Wrap

    • Wrap - flex-wrap
    • No wrap - flex-nowrap

  • Grow and shrink

    • Grow - flex-grow-1
    • No grow - flex-grow-0
    • Shrink - flex-shrink-1
    • No shrink - flex-shrink-0

Rounded corners

  • Custom radius - rounded-{radius}
    • radius is in pixels. It can be 0 or any multiple of 5 up to 30 (0, 5, 10, 15 ... 30) e.g.rounded-5
  • Circle - rounded-circle

Border

  • Around (1px) - border
  • No border - no-border
  • Custom width - b-{border-width}
    • border-width is in pixels. It can be anywhere from 0 to 10. e.g. b-5 produces a 5px border
  • Partial border - b{direction}-{border-width}
    • possible codes for direction are: t (top), b (bottom), l (left), r (right), x (left and right) and y (top and bottom) e.g. bt-5 produces a 5px border to the top

Margin and padding

  • Custom margin/padding width - m-{width} and p-{width}
    • width is in rem unit. It can be anywhere between 0 and 10. e.g. m-2 produces a 2 rem margin, p-2 produces a 2 rem margin
  • Partial margin/padding - m{direction}-{width} and p{direction}-{width}
    • possible codes for direction are: t (top), b (bottom), l (left), r (right), x (left and right) and y (top and bottom) e.g. mt-2 produces a 2 rem margin to the top, px-3 produces 3 rem paddings to the left and right sides.
